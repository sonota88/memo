== sbt

[source]
---------------------
sbt をダウンロードして展開
sbt のあるディレクトリにパスを通す
ソースは src/main/scala/ の下に置く

build.sbt 書く。内容は↓を参照。
sonota88/scala-cli-template
https://github.com/sonota88/scala-cli-template

sbt "run arg1 arg2" で実行
---------------------

== 条件分岐

=== if

[source,scala]
---------------------
if ( ... ) {
  // ...
} else if ( ... ) {
  // ...
} else {
  // ...
}

// 三項演算子の代わり
val s:String = if (isFoo) "true" else "false"
---------------------


== 反復

=== foreach

[source,scala]
---------------------
xs.foreach({x =>

  // ...
}
